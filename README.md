## 关于图标包的使用

- 自适应系统颜色及形状和明暗模式
- alterX 标准版 / 自适应系统形状 / 自适应系统颜色 / 明暗模式
- alterX Light / 自适应系统形状 / 颜色固定 / 白底黑图
- alterX Dark / 自适应系统形状 / 颜色固定 / 黑底白图
- Motorola / 三星 / 类原生 完全适用
- OPPO/ 一加 / 魅族 部分机型使用

- **如图标无法跟随系统变色请应用其他图标包后再次应用本图标包。如上操作仍无法更新图标请重启手机后再试。**
