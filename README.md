## 关于图标包的使用

* 本图标包共三个版本：


  * **alterX | 幻变图标包**

  * **alterX White | 幻变图标包×清朗**

  * **alterX Black | 幻变图标包×暗夜**

- alteX 图标包支持系统变色且支持深色模式。三星、OPPO、一加、摩托罗拉等手机均可使用。

- **如图标无法跟随系统变色请应用其他图标包后再次应用本图标包。如上操作仍无法更新图标请重启手机后再试。**

## 关于图标贡献

- 图标贡献请尽量使用矢量格式如AI、SVG、CDR等，尽量不要使用二进制图片。

- 位图请尽量上传PSD，如无PSD请尽量上传PNG、WEBP等格式且主体明确背景透明，图标主体大小416×416
